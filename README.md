## Currently Exploring

### Memoville

A diary app where short emotional records grow a living 3D village.

```text
DiaryEntry
  -> DiaryMemory
  -> VillageEffect[]
  -> VillageSnapshot
  -> Three.js Village
```

The goal is to make daily reflection feel light, visual, and alive.

### Role-Based AI Development Harness

A reusable method for using AI agents as a small product team instead of a
single code generator.

```text
planner
  -> domain-system-designer
  -> data-contract-engineer
  -> implementation-engineer
  -> qa-reviewer
```

## Working Style

- Prototype quickly, then turn the useful parts into systems.
- Keep product loops visible before adding features.
- Prefer warm, clear UX over noisy interfaces.
- Use AI as a collaborator with roles, handoffs, and review.
